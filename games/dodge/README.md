# Dodge for CHIP-8

## Introduction

Dodge is an action game. Operate Solvalou and avoid enemy characters. 

## How to Play

You can play [in your browser](https://johnearnest.github.io/Octo/index.html?key=mo04OUlO).

## How to Controll

Operate Solvalou with the [A] / [D] key or the [<-] / [->] key.
